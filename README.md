## Instruções do projeto

Depois de clonar o repositório, se é o seu primeiro clone do projeto use o seguinte comnado para baixar as dependências:
```node
npm i
```

Com tudo baixado corretamente, ja é possivel abrir o projeto em modo dev, pra isso use o seguinte comando:
```node
npm run dev
```
Com isso, o próprio Next.js atualizará dos erros e também atualizará o Tailwindcss
